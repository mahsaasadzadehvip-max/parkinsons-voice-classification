# 🧠 Parkinson’s Voice Classification

This project focuses on predicting Parkinson’s disease using voice measurements and statistical learning techniques.

## 📊 Project Overview

- Dataset: Voice recordings with biomedical features
- Goal: Classify healthy vs Parkinson’s patients
- Type: Supervised Learning (Classification)

## ⚙️ Methods Used

- Logistic Regression
- Decision Tree
- Data preprocessing & feature selection
- Model evaluation (accuracy, confusion matrix)

## 📈 Key Insights

- Certain voice features strongly correlate with Parkinson’s disease
- Logistic Regression provided interpretable results
- Decision Tree helped visualize classification logic

## 🛠️ Tools & Technologies

- Python
- Pandas
- Scikit-learn
- Matplotlib / Seaborn
